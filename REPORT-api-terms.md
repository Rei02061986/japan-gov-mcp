# 全省庁API利用規約調査報告書

**調査日**: 2026-03-03
**対象**: japan-gov-mcp で統合している全36 API
**目的**: クレジット表示義務、商用利用可否、プロキシ/再配布条件、禁止事項の網羅的調査

---

## 共通フレームワーク

日本政府APIの利用規約は主に以下の3つの枠組みに分類される。

| 枠組み | 策定年 | CC BY互換 | 商用利用 | 出典必須 |
|--------|--------|:---------:|:--------:|:--------:|
| **政府標準利用規約（第2.0版）** | 2015年 | Yes | Yes | Yes |
| **公共データ利用規約（PDL 1.0）** | 2024年 | Yes | Yes | Yes |
| **独自規約** | 各機関 | 個別 | 個別 | 個別 |

**共通ルール（政府標準利用規約/PDL 1.0）**:
- 出典記載形式: `出典：○○省ウェブサイト（当該ページのURL）`
- 加工時: `「○○」（○○省）（URL）を加工して作成`
- 禁止: 加工コンテンツを国が作成した未加工物のように表現すること
- CC BY 4.0 と互換（デュアルライセンス可能）

## 対応が必要なAPI

### 即時対応

1. **BOJ時系列API**: サービス公開済みのため、post.rsd17@boj.or.jp へ報告メール送信が必要
2. **全API**: レスポンスにクレジット/出典情報を付与する仕組みを実装

### 設計見直し検討

3. **J-SHIS**: 生データの直接返却を避け、要約・加工した形で提供
4. **J-STAGE**: 24時間キャッシュ制限を遵守するTTL設計
5. **ハローワーク**: MCPサーバーでの一般提供は規約上困難。利用可否の再検討

### 利用前の手続き

6. **JAXA**: 商用利用時にJAXAへ事前通知（earth@ml.jaxa.jp）
7. **NDLサーチ**: 営利利用時はNDLへ申請
8. **CiNii**: 商用利用時はNIIへ事前連絡
9. **researchmap**: JSTへの利用申請・承認取得
