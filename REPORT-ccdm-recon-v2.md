## 宿泊旅行統計（国籍×都道府県）の最終結論

### e-Stat API経由

- **国籍×都道府県の組み合わせは2014-2016の3年間（テーブル`0003317699`）のみ**
- 後継テーブルは存在しない
- テーブル`0000010107`で都道府県別の外国人延べ宿泊者数（総数）は2024年まで取得可能だが、フランス単独は不可（韓国・中国・台湾・米国の4国のみ）

## 訪日外国人消費動向の最終結論

- e-Stat APIでの取得可能期間: **2015Q4-2017Q3のみ**（テーブル`0003317184`、8四半期）
- 後継テーブル: **存在しない**（全517テーブル凍結、statsCode=00601030）
- 観光庁ウェブサイトにExcelファイルあり（tstat `000001079576`/`000001079655`、2024年まで）
- 調査名が「インバウンド消費動向調査」に改称された可能性あり

## JNTOデータの結論

- **e-Stat経由での取得: 不可**（JNTOは独自にデータ公表、e-Statにアップロードしていない）
- 検索キーワード「JNTO」「訪日外客数」「訪日外客統計」のいずれも0件

### 代替取得方法

**JNTOマスターExcelファイル（確認済み）**:
- URL: `https://www.jnto.go.jp/statistics/data/_files/20260218_1615-5.xlsx`
- 内容: 国籍/月別 訪日外客数（2003年1月〜2026年1月）
- サイズ: 437KB
- フランスを含む全国籍の月次データ
- **制約**: URLが日付スタンプ付きで変動する。一覧ページのスクレイピングが必要

**JNTOデータポータル（Tableau Public）**:
- URL: `https://statistics.jnto.go.jp/`
- 100+のビジュアライゼーション（都道府県別訪問者数含む）
- Tableau PublicのExport機能でCSV/Excel取得可能
- **制約**: Tableau APIでの自動取得には専用ツーリングが必要

## 結論：CCDMに使える最長時系列はこれ

### 推奨戦略（優先順位順）

#### 戦略A: API + Excelハイブリッド（推奨）

| 変数 | ソース | 期間 | 粒度 |
|------|--------|------|------|
| フランス人入国者数（全国） | e-Stat API `0003288044` | **1997-2024（年次28年）** | 全国×年次 |
| フランス人入国者数（月次） | e-Stat API `0003449064`+`0003287583` | **2013-2025（月次→四半期変換で~40Q）** | 全国×月次 |
| 都道府県別外国人宿泊者数 | e-Stat API `0000010107` (G7102) | **1975-2024（年次50年）** | 47都道府県×年次 |
| 都道府県×国籍別宿泊者数 | **観光庁Excel（確報）** | **2007-2024（推定、要Excel解析）** | 47都道府県×21国籍×年次 |
| 国籍別訪日外客数（月次） | **JNTO Excel** | **2003-2026（月次23年=92Q）** | 全国×国籍×月次 |

**この戦略で得られるフランス×四半期の最長時系列: ~92四半期（2003-2025）**
ただし全国レベル。都道府県分配にはExcelデータの解析が必要。

#### 戦略B: e-Stat APIのみ（Excel不要）

| 変数 | ソース | 期間 | 四半期換算 |
|------|--------|------|-----------|
| フランス人入国者数（月次→Q） | `0003449064`+`0003287583` | 2013-2025（ギャップ有） | **~40Q** |
| 都道府県別外国人宿泊者数 | `0000010107` (G7102) | 1975-2024 | 年次のみ |
| 都道府県×フランス宿泊者数 | `0003317699` | 2014-2016のみ | **12Q** |

**制約**: フランス×都道府県×四半期は12Qのみ（Granger検定には不十分）。
全国レベルなら40Q程度（2013-2025、ギャップ有）。

#### 戦略C: 推計パネル構築

1. 出入国管理統計（API）でフランス人入国者数の月次全国時系列（2013-2025、~40Q）
2. テーブル`0000010107`で都道府県別外国人宿泊シェア（年次2024まで）
3. テーブル`0003317699`でフランス人の都道府県配分比率（2014-2016固定）
4. 1×3の比率でフランス人の都道府県別推計値を算出
5. **推計精度**: 配分比率が時間的に安定していれば合理的

### 最終推奨

**戦略A（API + Excelハイブリッド）が最良**。

次のアクション:
1. 観光庁の確報Excel（2007-2024）をダウンロードして「参考第1表」の構造を確認
2. 国籍×都道府県データが含まれていればパーサーを構築
3. JNTOマスターExcelのパーサーも構築
4. BOJプロバイダの`db`パラメータバグを修正してEUR/JPY算出を可能に
5. これらをjapan-gov-mcpの新プロバイダ（`tourism.ts`）として統合

**期待されるCCDMパネル**: フランス人×47都道府県×四半期×~70Q（2007-2024）= **約3,290セル**
